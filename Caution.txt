2. 有以下程序:

main()
{
    int i=1;
    switch(i)
    {
	case 1:  printf("%d",i++);
	case 2:  printf("%d",i++);
	case 3:  printf("%d",i++);
	case 4:  printf("%d",i++);
	    break;
	default:  printf("END");
    }
}

3. 在C语言程序中,以下正确的描述是: B
