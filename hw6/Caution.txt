以下与 int *q[5]; 等价的定义语句是
###C###
    指针数组
	 (A)  int q[5] ;
	 (B)  int *q ;
	 (C)  int *(q[5]) ;
	 (D)  int (*q)[5] ;

若有说明: char *language[] = {"FORTRAN", "BASIC", "PASCAL", "JAVA", "C"};则以下不正确的叙述是
###D###
    二维数组相当于
    注意B和C
	   (A)  *(language+2) 表示字符串"PASCAL"的首地址
	   (B)  *language[2]的值是字母P
	   (C)  language是一个字符型指针数组,它包含5个元素,每个元素都是一个指向字符串的指针
	   (D)  language包含5个相同长度的数组
